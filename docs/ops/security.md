# Säkerhet

1. Checka aldrig in hemligheter. Håll .env.example uppdaterad.
2. Validera all input vid API-gränser.
3. Least-privilege API-nycklar.
4. Sanera användargenererat innehåll.
5. Rate-limita känsliga endpoints.
6. Logga säkerhetsrelevanta händelser.
7. Dokumentera behörigheter per integration.
8. Separera dev, staging och produktion.
